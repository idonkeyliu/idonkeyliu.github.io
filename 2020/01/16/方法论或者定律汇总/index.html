方法论或者定律汇总

发表于 2020-01-16 |

1. 为啥有方法论或者定律？

最开始的时候大家都是遇到事情然后想办法来解决问题，那么结果会有这么几个

  1. 问题没有得到解决；
  2. 问题得到了解决，但是解决问题的不是那么的好
  3. 问题得到了完美的解决。
    前者是失败，后两者是成功，总会有人分别总结失败与成功的经验，这样社会中就会有一些沉淀。久而久之，有些人会对成功的经验进行抽象，总结出其中共通的部分，那么这就是方法论或者定律，不过定律是更广泛的方法论。

2. 为啥要有方法论或者定律

吾生有涯而学海无涯，有涯的生命更需要追求成功率和效率。方法论或者定律就是提高工作效率和成功率的窍门。

3. 定律

3.1 二八定律

如果需要找出这世界普适的一条定律，那么我认为是二八定律。20%的XX带来80%的XX。典型的马太效应，强者俞强弱者愈弱。

3.2 熵增定律

封闭的系统，如果没有外力的作用，那么必然从有序走向无序。混乱是趋势，逆熵是需要自律的。

3.3 墨菲定律

担心发生的一定会发生。提醒我们一定要想周全。

3.4 彼得原理

雇员总会处于其不胜任的位置上

3.5 帕金森定律

大企业病，企业倾向越来越臃肿，人浮于事

3.6 晕轮效应

看喜欢的人啥都是优点，看不喜欢的人哪里都是缺点。

3.7 破窗效应

窗户如果破了一块玻璃，如果不及时修补，那么很快所有窗户都会被打破

3.8 首因效应

第一印象很重要，后续要改变需要花很大的功夫

3.9 电梯法则

一个事情需要在上电梯或者下电梯的这段时间能够讲解清楚，否则这个事情就是没有理解清楚

4. 常用的方法论

4.1 5W2H

设问的方式，5W2H可以将事情考虑的非常的全面
WHO:谁做
WHEN:什么时候
WHAT:是什么
WHY:为什么
WHERE:哪里
HOW:怎么做
HOW MUCH:程度

4.2 四象限法则

事情按照重要程度和紧急程度可以分为4个象限
重要紧急：救火的角色，会疲累
重要不紧急：最应该做的事情，而且需要规划好
不重要不紧急：交给别人做
不重要紧急：授权别人做
