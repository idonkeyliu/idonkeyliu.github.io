HTTP/2

HTTP/2是HTTP/1.1之后的HTTP传输标准，是下一代的HTTP协议。其优势有：

  1. 采用二进制格式传输数据，而HTTP/1.1是采用文本协议传输数据，所以在解析上更有优势
  2. 对消息头采用HPACK进行压缩传输，能够节省协议头所占的网络流量
  3. 多路复用，同一域名下的请求可以通过一个TCP连接并发完成，效率极大提高
  4. Server Push,服务器能够将资源传输给客户端

熵与反熵

熵是一个物理学概念，即一个系统混乱的程度。如果一个系统越无序，那么它的熵就越高，反之就越低。如果系统更加混乱了，就熵增，反之就熵减。而且无外力作用的情况下，系统一般会变得越来越无序，即熵越来越大。

熵增发生在封闭的系统内，如果是一个流动的开放系统，学名叫“耗散结构”，会产生负熵，这样系统会从无序走向有序。

流动是对抗熵增的关键，熵越大系统越不可治理，所以要达到治理的状态，一定需要反熵。
