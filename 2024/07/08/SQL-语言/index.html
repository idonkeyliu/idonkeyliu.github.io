SQL 语言

发表于 2024-07-08 |

1986 年，SQL（结构化查询语言）成为标准。在接下来的 40 年里，它成为关系数据库管理系统的主导语言。

1. SQL 简介

SQL 语言有 5 个组成部分：
DDL(Data Definition Language)：数据定义语言，例如 CREATE、ALTER、DROP
DQL(Data Query Language)：数据查询语言，例如 SELECT
DML(Data Manipulation Language)：数据操作语言，例如 INSERT、UPDATE、DELETE
DCL(Data Control Language)：数据控制语言，例如GRANT、REVOKE
TCL(Transaction Control Language)：事务控制语言，例如COMMIT、ROLLBACK
