学习《五星级沟通》

发表于 2018-12-19 |

进公司六年多了，刚开始的时候参加过一些培训课程，但是感觉收获都不是很大，可能我自己不太适合培训这种方式吧，所以渐渐的都不去参加了。公司有两门非常热门的课程叫《五星级沟通》与《金字塔思维》，陆陆续续听到有些同事去听这两门课程，搞得自己也忍不住想去听一下，所以就都报名了。因为太火爆了，所以根据报名情况进行排队在，直到今天才去学习了《五星级沟通》。下面对今天的学习进行总结下，也算是复习了。

人最重要的属性是社会性，社会性就是指人不是孤立存在的，他是需要与周围的人交流互动进而产生一定关系的。所以沟通就是最常见也最必不可少的方式了。参加课程前我只知道沟通的学问很多，因为无论在生活中还是在工作中，总会有人沟通起来很顺畅，有些人沟通起来很费劲，这里面一定有些门道。那怎么样才能做好沟通呢？

1. 建立沟通意识

沟通是最常见，最容易被忽视的一种行为，因为大家大多数时间都是在做沟通，不自觉的沉溺其中而不知道自己做的有哪些不够。

1.1 建立沟通意识

戒掉以自我为中心的沟通，建立换位思考的沟通方式

1.2 沟通的重要性

专家研究说20%的沟通是有效沟通，80%的是无效沟通，虽然这个数据不一定靠谱，但是想想看也觉得我们沟通过程中确实存在着很多问题。说的激烈一点，我们很多时间都是在说废话，在做无效沟通要么是浪费别人的时间，要么是让别人浪费我们的时间。

1.3 明确沟通目的

有效的沟通从明确目的开始，有些人讲了半天别人都不知道他想干嘛。目的不明，重点丢失。沟通的实质是达到目的，所以明确目的对于沟通来说非常重要。不明确沟通目的的沟通注定是效率低下甚至是无效的。

2. 明确沟通的障碍

2.1 人的因素

比如认知能力、情绪、有选择的接受等。

2.2 人际关系

沟通双方的了解与信任、沟通双方的相似性等。

2.3 组织结构

上下级、传递链条等

2.4 技术因素

语言、信息过载、信息丢失

2.5 客观环境

嘈杂的环境、不适合的时机、人为干扰等。

3. 锤炼技能

3.1 说

先说事实，再说主观感受，最后加以适当的表演形式，这样说才具备系统性与逻辑性。只有基于事实才能作出最有成效的判断与抉择。有的人说话的时候只加自己主观的感受，我觉得、我以为等，这样的说辞毫无说服力。下面是说的评估标准

评估点 标准
目的选择 目的传递清晰
符合逻辑 符合框架
受众分析 切入每个听众的利益点
动态反馈 及时根据听众的情况进行内容调整
不跑题 发散出去后可以收的回来

表达的效果：7%在于你说的什么 38%在于你的语音语调 55%在于你的肢体语言，这点老外做的比我们好。

3.2 听

一个好的倾听者总是非常的难得。人总是喜欢表达自己的观点，而不是倾听。运营ASH原则就可以做一个好的倾听者，A即是接受情绪，S即是分享感受，H即是提供支持。接受情绪，能够拉近与说话人的距离，让他觉得你与他是在同一战线的；分享感受能进一步的拉近距离，让说话的人觉得你可以感同身受；光有接受情绪与分享感受还不够（当然有些时候别人就只是需要一个倾听者而已，哪怕这个倾听者啥都不说），说的人是需要达到某种目的的，那么听的人最好能提供某些建设性的意见，这样就能很好的作为一个倾听者了。

3.3 问

只听不说不是好听众，沟通不是单向的，有时候适当的互动会让沟通更加的有效，能让说话的人觉得你是在全程follow。好的提问者一定非常深刻，今天还学到了一个名词，苏格拉底式的提问方式，就在于不断的发问，引导说者不断地深入思考。提问分开放式提问与封闭式提问，在具体的场景下可以做合适的选择。另外有些提问是试探，是为了考察你思考的深度与高度。

4. 厘清对象

对象分很多种，比如上级、平级、下级、跨部门、跨BG、合作伙伴、客户等，对不同的对象需要有不同的沟通方式与手段。沟通方式的有效性依次为面对面、电话、即时通讯工具、邮件。

5. 巩固关系

巩固关系就是巩固影响力，从而提高你的能力。降低无效沟通与无效社交。

最后

上述说的都是套路，更多的还是要善良。真诚待人，同理心出发，相信沟通不是一件难事。
